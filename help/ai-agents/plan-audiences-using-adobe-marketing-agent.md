# Plan audiences using Adobe Marketing Agent for Microsoft 365 Copilot

Learn how Adobe Marketing Agent for Microsoft 365 Copilot simplifies audience planning directly within Microsoft Teams. This tool lets marketers identify loyalty audiences, check journey lineage, and validate campaign candidates without switching platforms or filing requests with operations teams.

This video demonstrates the process of selecting, analyzing, and preparing audiences for campaigns. It showcases how the agent generates actionable insights, surfaces related suggestions, and provides lineage visualization between audiences and journeys. Marketers can efficiently manage data and streamline workflows, ensuring focused targeting and improved campaign outcomes.
