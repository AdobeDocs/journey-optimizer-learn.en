# Create omnichannel content from scratch

In this step you start from an existing content template and generate a full asset — such as an email, push notification, or in-app message — using AI capabilities. You apply brand guidelines and configure language, tone, and image settings to keep content on brand. You generate multiple AI-driven variations, review brand scoring, and edit content by hand or with a second AI pass to refine messaging and visuals.
