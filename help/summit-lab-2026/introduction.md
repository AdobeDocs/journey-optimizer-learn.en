# AI-Powered Decisioning for Web Experiences in Adobe Journey Optimizer - Introduction

Originally presented at Adobe Summit 2026 as Lab L610, this tutorial guides you through the lab's content.

Over the course of five exercises, you learn how to construct a decisioning system for SecurFinancial using Adobe Journey Optimizer. You configure logic that identifies the most suitable product offer per customer based on their financial profile and real-time actions, then deploy it consistently via a web banner and SMS.
