# Create web push notifications

Web Push allows websites to send notifications directly to a user's device, even when the browser is in the background. Learn how to create and send Web Push Notifications in Adobe Journey Optimizer. This tutorial guides you through selecting channel configurations, authoring messages, choosing audiences, and publishing campaigns to expand your reach beyond mobile devices.
