# Set up schemas and datasets for campaigns

Learn how to set up relational schemas and datasets for orchestrated campaigns in Adobe Journey Optimizer. This guide covers defining keys, relationships, and using DDL files to create schemas efficiently. You'll also learn about ingesting data via CSV uploads and Change Data Capture (CDC), validating datasets, and preparing them for querying, personalization, and campaign execution. Watch the embedded video for step-by-step instructions to streamline your campaign workflows.
