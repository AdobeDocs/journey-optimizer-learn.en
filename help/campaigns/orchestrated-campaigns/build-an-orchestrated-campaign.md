# Build an orchestrated campaign

Learn how to design and execute orchestrated campaigns in Adobe Journey Optimizer by building workflows that combine audiences, data manipulation, and multi-channel delivery. This video walks through campaign scheduling, audience creation from relational data, enrichment and deduplication, and using multiple streams to support advanced use cases like abandoned cart follow-ups, churn prevention, and back-in-stock notifications.
